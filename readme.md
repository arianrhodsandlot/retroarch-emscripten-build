# RetroArch Emscripten build

This repository contains RetroArch builds for emscripten.

Files are all grabbed from [Libretro Buildbot](https://buildbot.libretro.com/stable/), for example, https://buildbot.libretro.com/stable/1.20.0/emscripten/RetroArch.7z, and then extracted without any modification.

The ScummVM core was removed from this repository since v1.20.0 as it's too big (100M+, which is far larger than a reasonable size for web).

We can use jsDelivr, a public CDN that allows us to access files from GitHub, to load these files. Here are a full list of contained files, with their jsDelivr CDN URLs.
